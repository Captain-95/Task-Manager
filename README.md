\# Task Manager



A full-stack Task Management application built as an interview assignment.



\## Tech Stack



\### Backend

\- Java 17

\- Spring Boot

\- Spring Security

\- JWT Authentication

\- H2 Database



\### Frontend

\- Angular

\- Angular Material

\- Chart.js

\- Responsive UI



\## Features



\- JWT Login

\- Role Based Authentication (Admin/User)

\- Dashboard

\- Task CRUD

\- Dashboard Charts

\- Search \& Filter

\- Responsive Design
